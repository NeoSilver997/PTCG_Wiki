# Copilot - PTCG 卡牌資料庫 (繁體中文版)

## 概述

這是一個現代化的 PTCG (寶可夢卡牌遊戲) 卡牌資料庫的 HTML 模擬頁面集合，採用深色主題設計，搭配青色/藍綠色漸層色彩配置。所有頁面均使用繁體中文介面，並整合了互動式功能。

## 設計特色

### 🎨 視覺設計
- **深色主題**：採用 slate-900 作為主背景色，創造現代化的深色體驗
- **青色漸層**：使用 teal-400 到 cyan-500 的漸層色彩作為主色調
- **毛玻璃效果**：部分元件採用 backdrop-blur 效果，增加視覺層次
- **動畫效果**：包含浮動動畫、懸停效果和轉場動畫

### 📱 響應式設計
- 支援桌面、平板和手機等各種螢幕尺寸
- 使用 Tailwind CSS 的響應式工具類
- 網格布局適應不同螢幕寬度

### 🏗️ 頁面結構

#### 1. 首頁 (index.html)
- **砌塊布局**：採用 CSS Grid 實現的卡牌砌塊展示
- **搜尋功能**：即時卡牌搜尋和篩選
- **統計面板**：展示資料庫統計資訊
- **精選卡牌**：突出顯示熱門卡牌

#### 2. 管理頁面 (admin.html)
- **儀表板設計**：展示系統統計和快速操作
- **圖表元件**：使用簡易的進度條和統計圖
- **管理功能**：卡牌管理、使用者管理和系統設定

#### 3. 卡牌編輯 (admin-card-edit.html)
- **分頁介面**：使用 JavaScript 實現的分頁切換
- **表單驗證**：互動式表單驗證和即時預覽
- **圖片上傳**：支援拖拽上傳和預覽功能

#### 4. 卡牌詳情 (card-detail.html)
- **詳細資訊展示**：完整的卡牌資訊展示
- **相關卡牌推薦**：相似卡牌的橫向滾動列表
- **價格歷史**：價格走勢圖表

#### 5. 收藏管理 (collection.html)
- **卡牌網格**：可排序和篩選的卡牌集合
- **批量操作**：支援多選和批量操作
- **收藏統計**：價值統計和收藏分析

#### 6. 牌組建構器 (deck-builder.html)
- **拖拽功能**：支援拖拽卡牌到牌組列表
- **即時驗證**：牌組規則驗證和建議
- **牌組分析**：屬性分布和勝率分析

#### 7. 賽事列表 (events.html)
- **砌塊布局**：使用 CSS Columns 實現的賽事砌塊
- **篩選器**：多重條件篩選賽事
- **統計卡片**：賽事統計資訊展示

#### 8. 賽事詳情 (event-detail.html)
- **英雄區塊**：大型賽事資訊展示
- **側邊欄**：報名資訊和相關賽事
- **即時戰況**：賽事進行中的即時更新

#### 9. 登入頁面 (login.html)
- **動態背景**：浮動形狀動畫背景
- **分頁切換**：登入和註冊表單切換
- **社交登入**：支援第三方登入選項

## 技術實現

### 前端框架
- **Tailwind CSS**：用於樣式設計和響應式布局
- **Google Fonts (Noto Sans TC)**：繁體中文字體支援
- **Vanilla JavaScript**：實現互動功能和動態效果

### 互動功能
- **分頁切換**：使用 JavaScript 實現的分頁介面
- **拖拽上傳**：支援檔案拖拽上傳功能
- **表單驗證**：即時表單驗證和錯誤提示
- **搜尋篩選**：即時搜尋和多條件篩選

### 設計模式
- **元件化設計**：重複使用的 UI 元件
- **一致的色彩系統**：統一的色彩配置
- **親和的互動設計**：直觀的使用者介面

## 檔案結構

```
Copilot/
├── index.html              # 首頁 - 砌塊布局卡牌展示
├── admin.html              # 管理頁面 - 系統儀表板
├── admin-card-edit.html    # 卡牌編輯 - 分頁表單介面
├── card-detail.html        # 卡牌詳情 - 詳細資訊展示
├── collection.html         # 收藏管理 - 卡牌網格管理
├── deck-builder.html       # 牌組建構器 - 拖拽建構介面
├── events.html             # 賽事列表 - 砌塊布局賽事展示
├── event-detail.html       # 賽事詳情 - 完整賽事資訊
├── login.html              # 登入頁面 - 動態背景設計
└── README.md              # 專案說明文件
```

## 使用說明

1. **開啟頁面**：直接在瀏覽器中開啟任一 HTML 檔案
2. **導航操作**：使用頂部導航列在不同頁面間切換
3. **互動功能**：點擊按鈕、拖拽元素體驗各種功能
4. **響應式測試**：調整瀏覽器視窗大小測試響應式設計

## 設計理念

這個 Copilot 版本的設計理念是創造一個現代化、深色的 PTCG 卡牌資料庫介面，強調使用者體驗和視覺美觀。通過深色主題和青色漸層的搭配，創造出科技感和專業感的視覺效果，同時保持良好的可讀性和操作性。

## 相容性

- **瀏覽器支援**：Chrome 80+, Firefox 75+, Safari 13+, Edge 80+
- **螢幕解析度**：支援 320px 以上寬度的裝置
- **網路需求**：需要網路連線載入 Tailwind CSS 和 Google Fonts
