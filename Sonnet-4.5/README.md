# PTCG CardDB - Claude Sonnet 4.5 Edition

## 🎨 設計主題
**深海夜空 × 琥珀金輝 (Navy Midnight × Amber Gold)**

專為 Claude Sonnet 4.5 打造的專業級賽事分析介面，結合深邃的海軍藍/石板灰漸層背景與溫暖的琥珀金色點綴，呈現出優雅、專業且具有科技感的視覺體驗。

### 配色方案
- **主背景**: `linear-gradient(135deg, #1e293b 0%, #0f172a 100%)` - 深海軍藍到午夜黑的漸層
- **玻璃面板**: `rgba(51, 65, 85, 0.6)` - 半透明石板灰，搭配 12px 模糊效果
- **金色強調**: `linear-gradient(135deg, #fbbf24 0%, #f59e0b 100%)` - 琥珀金到深金的漸層
- **文字色彩**:
  - 主標題: `#fbbf24` (琥珀金) + 發光效果
  - 次標題: `#fcd34d` (淺金色)
  - 正文: `#e2e8f0` (淺灰白)
  - 輔助文字: `#94a3b8` (石板灰)

### 特色元素
- **發光效果 (stat-glow)**: 金色數據搭配 `text-shadow: 0 0 10px rgba(251, 191, 36, 0.5)` 打造科技感光暈
- **玻璃擬態 (Glassmorphism)**: 磨砂玻璃背景 + 細邊框營造層次感
- **優雅動畫**: 卡片懸停時平滑放大 + 金色陰影強化互動回饋
- **專業圖標**: ✦ (菱形星) 作為品牌標誌，呼應「菁英」定位

## 🎯 設計理念

### 專業級定位
以 **Claude Sonnet 4.5** 的高階 AI 能力為靈感，打造符合專業玩家需求的介面:
- **數據驅動**: 使用率、勝率、獎金池等核心指標一目了然
- **視覺層次**: 玻璃擬態 + 金色點綴營造科技感與專業度
- **互動流暢**: 懸停動畫、發光效果提升操作回饋

### 色彩心理學
- **深藍/黑背景**: 專注、深度、專業
- **琥珀金強調**: 價值、高端、菁英
- **玻璃質感**: 透明、先進、未來感

### 響應式佈局
- 最大寬度: 1920px (7xl 容器)
- 卡片網格自適應 (3-6 欄)
- 固定導航欄 (sticky top-0)

## 🚀 技術棧

- **框架**: Tailwind CSS 3.x (CDN)
- **字型**: Noto Sans TC (Google Fonts)
- **圖標**: Unicode Emoji + SVG
- **動畫**: CSS Transitions + Keyframes
- **樣式特效**:
  - `backdrop-filter: blur(12px)` - 磨砂玻璃
  - `text-shadow` - 金色發光效果
  - `transform: translateY()` - 懸停抬升
