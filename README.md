# PTCG CardDB - 靜態 UI 原型

寶可夢卡牌資料庫的靜態 HTML/CSS 原型 (繁體中文)

## 📁 文件結構

- `index.html` - 卡片列表頁面 (主頁)
- `card-detail.html` - 卡片詳情頁面
- `collection.html` - 我的收藏頁面
- `login.html` - 登入頁面
- `script.js` - 基本互動功能

## 🎨 設計特點

### 語言
- **預設語言**: 繁體中文 (zh-HK)
- 支援未來英文版本 (語言切換器已預留)

### 佈局
- **桌面優先**: 最小寬度 1280px
- 6 欄卡片網格
- 左側篩選邊欄
- 固定頂部導航

### 顏色主題
- **主色**: 藍色 (#2563eb) - Pokemon 官方藍
- **類型顏色**: 
  - 火 🔥 - 紅色/橙色
  - 水 💧 - 藍色/青色
  - 草 🌿 - 綠色
  - 電 ⚡ - 黃色
  - 超 🔮 - 紫色
  - 鬥 👊 - 琥珀色
  - 妖精 🎀 - 粉紅色
  - 無 ⚪ - 灰色

### 技術棧
- **CSS Framework**: Tailwind CSS (CDN)
- **字體**: Noto Sans TC (Google Fonts) - 繁體中文優化
- **JavaScript**: Vanilla JS (無框架)

## 📄 頁面說明

### 1. 首頁 / 卡片列表 (`index.html`)
- 24 張模擬卡片數據
- 左側篩選器:
  - 搜尋框
  - 擴充包篩選 (SV08, SV07, SV06, SV05)
  - 類型篩選 (8 種類型)
  - 稀有度篩選 (UR, SR, RR, R, U, C)
- 6 欄響應式網格
- 分頁導航

### 2. 卡片詳情 (`card-detail.html`)
- 左側: 大型卡片圖片 + 操作按鈕
- 右側: 卡片詳細資訊
  - 卡片名稱 (中文/英文)
  - 編號、擴充包、類型、HP
  - 弱點、撤退費用
  - 招式說明
- 分頁標籤:
  - 價格資料 (多個來源比價)
  - 語言版本 (繁中/日文/英文)
  - 相關卡片

### 3. 我的收藏 (`collection.html`)
- 統計卡片:
  - 總卡片數: 245
  - 完成度: 67%
  - 總價值: HK$ 12,450
  - 稀有卡片: 18
- 篩選和排序功能
- 卡片網格顯示擁有數量徽章
- 每張卡片顯示市場價值

### 4. 登入頁面 (`login.html`)
- OAuth 登入選項:
  - Google
  - Apple
  - GitHub
- 傳統電郵/密碼登入
- 註冊連結
- 服務條款/隱私政策連結

## 🚀 使用方法

1. 直接在瀏覽器開啟任何 HTML 文件
2. 建議從 `index.html` 開始瀏覽
3. 無需本地伺服器 (純靜態文件)

## 📊 模擬數據

### 卡片資料:
- 24 張不同寶可夢卡片
- 涵蓋所有類型和稀有度
- 擴充包: SV05-SV08
- 卡片編號: 格式 `SV0X XXX/108`

### 收藏數據:
- 12 張擁有的卡片
- 數量從 1-4 張不等
- 總價值約 HK$ 12,450

### 價格數據:
- Yuyu-tei (日本)
- Hareruya (日本)
- CardMarket (歐洲)
- 顯示庫存和更新時間

## 🎨 設計系統

### 字體大小:
- H1: 3xl (1.875rem)
- H2: 2xl (1.5rem)
- H3: xl (1.25rem)
- 正文: base (1rem)
- 小字: sm (0.875rem)
- 極小: xs (0.75rem)

### 間距:
- 容器邊距: px-6
- 卡片間距: gap-4
- 區塊間距: mb-6, mb-8

### 圓角:
- 按鈕: rounded-md (0.375rem)
- 卡片: rounded-lg (0.5rem)
- 徽章: rounded-full

### 陰影:
- 卡片: shadow-md
- 懸停: shadow-lg
- 模態框: shadow-2xl
