# Bard - PTCG 卡牌資料庫模擬介面

## 概述

這是 PTCG 卡牌資料庫的 Bard 版本模擬介面，使用紅色/橙色主題設計，採用繁體中文介面。這個版本專注於提供完整的 PTCG 卡牌管理、比賽追蹤和牌組建構功能。

## 設計特色

### 視覺主題
- **主色調**: 紅色到橙色的漸層設計 (`from-red-600 to-orange-600`)
- **輔助色彩**: 橙色、琥珀色和黃色系
- **設計風格**: 現代化、專業的卡牌遊戲介面

### 介面語言
- **主要語言**: 繁體中文 (zh-TW)
- **字體**: Noto Sans TC (Google Fonts)
- **本地化**: 完整的繁體中文介面翻譯

## 檔案結構

```
Bard/
├── index.html          # 首頁 - 卡牌瀏覽和搜尋
├── admin.html          # 管理員面板 - 系統管理
├── admin-card-edit.html # 卡牌編輯介面 - 卡牌管理
├── card-detail.html    # 卡牌詳情頁面 - 單張卡牌資訊
├── collection.html     # 收藏管理 - 使用者卡牌收藏
├── deck-builder.html   # 牌組建構器 - 牌組建立工具
├── events.html         # 賽事列表 - 比賽和錦標賽
├── event-detail.html   # 賽事詳情 - 單場比賽資訊
├── login.html          # 登入頁面 - 使用者認證
└── README.md          # 專案說明文件
```

## 功能特色

### 1. 卡牌管理系統
- **卡牌瀏覽**: 支援篩選、搜尋和排序
- **卡牌詳情**: 完整的卡牌資訊展示
- **收藏管理**: 使用者個人卡牌收藏
- **價格追蹤**: 即時卡牌價格資訊

### 2. 牌組建構工具
- **拖拽介面**: 直覺的卡牌拖拽建構
- **牌組驗證**: 自動檢查牌組規則
- **匯出匯入**: 支援牌組檔案操作
- **統計分析**: 牌組強度分析

### 3. 賽事追蹤系統
- **賽事列表**: 多種賽事類型篩選
- **賽事詳情**: 完整的比賽資訊
- **參賽者管理**: 選手報名和資訊
- **即時更新**: 比賽結果追蹤

### 4. 管理員功能
- **卡牌管理**: 新增、編輯和刪除卡牌
- **使用者管理**: 帳號和權限管理
- **系統設定**: 全域設定調整
- **資料匯出**: 系統資料匯出功能

- **卡片佈局**: 資訊分塊展示

### 互動功能
- **表單驗證**: 即時輸入驗證
- **拖拽操作**: 牌組建構拖拽功能
- **動態篩選**: 即時搜尋和篩選
- **頁面切換**: 無刷新頁面導航
