# Claude-2 PTCG Mockup Set

## Overview

This is the Claude-2 mockup set for the PTCG CardDB project, featuring a blue/indigo gradient theme with Traditional Chinese localization. The design emphasizes tournament results and deck management data, using sample data featuring "莉佳的霸王花ex" (Rika's Vaporeon ex).

## Design Theme

- **Color Scheme**: Blue to indigo gradient backgrounds
- **Typography**: Noto Sans TC for Traditional Chinese text
- **Focus**: Tournament results, deck data, and performance metrics
- **Layout**: Data-heavy cards with tournament statistics and deck breakdowns

## Files Included

### Core Pages
- `index.html` - Tournament results homepage with featured decks
- `admin.html` - Administration dashboard with tournament management
- `admin-card-edit.html` - Tournament result and deck editing interface
- `card-detail.html` - Detailed card view with tournament performance data
- `collection.html` - Collection management with tournament deck focus
- `deck-builder.html` - Interactive deck building with drag-and-drop functionality
- `events.html` - Tournament listings with filtering and search
- `event-detail.html` - Detailed tournament view with standings and prize pools
- `login.html` - Authentication page with social login options

## Key Features

### Tournament Results Focus
- Live tournament standings and rankings

### Deck Management

- Performance charts and graphs

## Technical Implementation

- **Framework**: Pure HTML5, Tailwind CSS, JavaScript
- **Styling**: Responsive design with gradient backgrounds
- **Interactivity**: Form validation, drag-and-drop, dynamic filtering
- **Localization**: Full Traditional Chinese interface
- **Accessibility**: Semantic HTML and keyboard navigation
