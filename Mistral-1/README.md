# Mistral-1 PTCG Mockup Set

## 概述

Mistral-1 為輕柔、柔和色系的 mockup 集，風格採用「軟式 (soft / neumorphism)」視覺，內容以繁體中文呈現，專注於：

- 賽事結果總覽（`index.html`）
- 牌組總覽（`deck-builder.html`）
- 使用樣本：**莉佳的霸王花ex**

## 設計重點

- 色調：溫暖漸層與柔和陰影，強調「閱讀舒適」與「數據清晰」
- 字體：Noto Sans TC
- 目標：快速檢視賽事結果與單一牌組的比賽表現

## 使用方式

1. 開啟 `index.html` 查看賽事結果與牌組焦點
2. 點擊「查看牌組」或直接開啟 `deck-builder.html` 查看牌組明細
