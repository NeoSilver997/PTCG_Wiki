# Raptor-mini PTCG Mockup Set

## 概述

Raptor-mini 是一組以深色、精簡且數據導向為核心的 mockup，採用繁體中文，範例資料使用 **莉佳的霸王花ex**。此套重點在於清楚展示賽事結果與牌組資訊，並提供快速查看與匯出功能。

## 包含檔案

- `index.html` — 賽事結果總覽（暗色主題，重點數據卡與結果表格）
- `deck-builder.html` — 牌組檢視與編輯（牌組摘要、卡片清單、備註、驗證）
- `README.md` — 本檔案，含設計說明

## 設計重點

- 主題：深色調 + 青綠強調色
- 版面：資料導向、簡潔卡片與表格
- 互動：預留匯出/驗證按鈕與備註欄位，方便擴充拖放或圖表
