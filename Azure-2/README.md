# Azure-2 PTCG Mockup Set

## 概述

Azure-2 是一個以 **青綠 / 玻璃風 (glassmorphism)** 為主題的 mockup 集，採用 **繁體中文**，專注於「賽事結果」與「牌組總覽」兩個頁面，並以樣本資料 **莉佳的霸王花ex** 作為展示重點。

## 含檔案

- `index.html` — 賽事結果總覽（近期結果表格、快速檢視、樣本牌組總覽）
- `deck-builder.html` — 牌組總覽（牌組摘要、主要卡片清單、最近比賽成績）
- `README.md` — 此檔案，包含設計與使用說明

## 設計要點

- 顏色：青綠與天藍漸層搭配半透明玻璃卡片
- 字體：Noto Sans TC（繁體中文）
- 目標：以簡潔、資料導向的方式呈現賽事結果與牌組表現，方便快速閱覽與分享

## 使用方式

1. 打開 `index.html` 瀏覽賽事結果與樣本牌組表現
2. 在 `deck-builder.html` 查看牌組明細、驗證與匯出選項
